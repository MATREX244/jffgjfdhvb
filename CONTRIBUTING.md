# Contribuindo para o MalScan

Obrigado por seu interesse em contribuir para o MalScan! Como um projeto focado em segurança, valorizamos contribuições que melhorem a precisão da detecção e a robustez do sistema.

## Como Contribuir

1. **Relatar Bugs**: Abra uma Issue descrevendo o problema, passos para reproduzir e o comportamento esperado.
2. **Sugerir Melhorias**: Adoramos novas ideias! Sinta-se à vontade para sugerir novas assinaturas de malware ou melhorias na interface.
3. **Pull Requests**:
   - Faça um fork do projeto.
   - Crie uma branch para sua feature (`git checkout -b feature/NovaAssinatura`).
   - Faça o commit de suas alterações.
   - Envie para a branch (`git push origin feature/NovaAssinatura`).
   - Abra um Pull Request.

## Padrões de Código

- Siga as convenções de TypeScript.
- Garanta que o código esteja formatado com Prettier.
- Adicione comentários em partes complexas do motor de análise.

## Segurança

Se você encontrar uma vulnerabilidade de segurança no MalScan, por favor, não abra uma issue pública. Envie um e-mail para o mantenedor do projeto.
